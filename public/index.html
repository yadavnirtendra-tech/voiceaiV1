🏗️ Architecture

🔐 Zero-Friction Auth

Unified OAuth 2.0 flow. AES-256-GCM encrypted token storage with automatic refresh rotation.

📡 Webhook Listeners

Google Push Notifications + Microsoft Delta Queries. Sub-second sync latency instead of polling.

🛡️ Shadow Blocking

Automatic "Reserved" events injected across providers with infinite loop prevention via metadata tags.

⚡ Conflict Resolution

Configurable strategies: Block All, Priority-Based, or Manual. Global availability truth across all calendars.
